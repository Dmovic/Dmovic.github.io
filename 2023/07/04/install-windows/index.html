重装系统win10

Posted on 2023-07-04

包含硬盘、操作系统安装、基本软件安装

硬盘

由于本机硬盘被划分为了D、E、F等盘，一开始想要通过win11自带的恢复功能将电脑恢复为初始化状态（后来失败了，找不到恢复环境），所以首先将硬盘进行格式化并合并为只有一个C盘。

恢复分区的删除

磁盘管理工具中发现在C盘和DEF盘中间有恢复分区，无法合并，需要删除恢复分区，windows敲出cmd窗口。

进入硬盘分区工具

查看已有的硬盘

选择硬盘

列出所选硬盘的分区

选择要删除的恢复分区

删除恢复分区

将硬盘合并

恢复分区删除后，可以将磁盘进行合并。

此电脑-管理-磁盘管理中查看详细情况，分区合并时只能从后往前合并

  • 点击后面的被合并的分区，删除卷。本来蓝色的分区会变成灰色的未分配
  • 点击前面的要合并的分区，扩展卷，将后面的未分配合并进来

操作系统

采用u盘介质安装

u盘格式化

需要一个8G以上的u盘。格式化时可能坑比较多，经常出现windows未能格式化，需要耐心多重复几次。

u盘插入电脑，右键选择格式化，文件系统选择NTFS，分区选择默认大小，取消快速格式化，点击后等待。这里使用的是金士顿kingston的16G的u盘，失败了需要继续尝试。

软件下载

找到win10的微软官方下载，https://www.microsoft.com/zh-cn/software-download/windows10

其中选择立即下载工具，下载到格式化好的u盘中

此时最新的win10版本是22h2

运行下载工具

以管理员身份运行下载工具，选择u盘介质，并选择该u盘。期间会自动进行下载和安装后的清理。

注意在制作u盘介质的过程中，很可能会报错，各种代码，注意检查

  • u盘是否正确格式化
  • 下载工具是否放在u盘中
  • 是否使用管理员身份运行

制作成功后，工作已经完成了大半

重新安装操作系统

需要查看对应型号的设置，联想是F12。保持u盘插入，开机后疯狂按F12，进入选项，选择从u盘启动，顺利的话会进入安装程序，依次操作即可。

选择完毕后会自动重启，重启时立即拔下u盘，避免再次从u盘启动。

将进入windows的初始化设置界面，设置用户名、密码等

基本软件

此时完成开机，能看到清爽的桌面。

对于笔记本已经自带的正版windows激活，联网之后会自动读取主板上的序列进行激活，在系统中查看到已激活。

一般是家庭版，可以参考b站升级为专业版。笔者尝试过升级成专业版来暂停自动更新，但专业版中没有找到一键的操作，后来也就还是用的家庭版。

家庭版中可以通过服务-Windows Update等操作暂停

office tool

可以通过密钥下载基础的office套装，但如果只安装一次软件，可以考虑office tool，安装的企业版Microsoft 365
